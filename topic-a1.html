Getting Input

So you have a basic setup. Now What?

We do what generations of programmers have done before us: we make a Hello World program.

After creating a basic setup on Eclipse, you should have a Java Class file that looks something like this:

public class Main{
	public static void main(String[] args){
		//TODO
	}
}

We will learn more about what each of these mean later in the course. For now, let's focus on outputting Hello World! to the console

Type this code: System.out.println("Hello World!"); under the //TODO line. If you don't understand what you're typing, don't worry! We will learn more about it later in the course.

For now, your file should look like this: 

public class Main{
	public static void main(String[] args){
		//TODO
		System.out.println("Hello World!");
	}
}
Note: Don't forget the semicolon at the end of the line!

If you run this program in your IDE, you should see: Hello World! in the output.

Ok, that's cool. Now, what if we want to print more complicated messages? let's say, many lines of output. We could try copy-pasting the System.out.println("Hello World!") multiple times, replacing "Hello World!" with a new line of our message each time: 

public class Main{
	public static void main(String[] args){
		//TODO
		System.out.println("Line 1: to be or not to be");
		System.out.println("Line 2: that is the question");
		System.out.println("Line 3: - Shakespeare");
	}
}

This would work if you ran it in your compiler, but that's a lot of work. Is there a simpler way to do this?

The answer, of course, is Yes! We can use this sequence of letters: \n (backslash followed by lowercase letter n) This character signifies for java to start a new line in the output. Take this example: 

public class Main{
	public static void main(String[] args){
		//TODO
		System.out.println("Line 1: to be or not to be\nLine 2: that is the question\nLine 3: - Shakespeare");
	}
}

This will yield the same output as the previous block of code, however this version is a bit more concise.

Here's a tricky situation. What will the following code output: 

System.out.println("Line 1:\n Line 2:\nLine 3:");
The tricky bit to notice is that Line 2 will be preceeded by a space. Be careful about placing spaces in output, as they might alter the formatting of your output.

Declaring Variables

A very important feature of Java, and most programming langauges, are variables. We will use variables a lot in the rest of this course, so make sure to pay attention to this section.

To understand variables, consider the following example: 

int x = 0;
In this code, we declare a variable of type int, name it x, and assign it the value of 0.

We can assign x any value we want, as long as it fits the constraints of its datatype

For example: we can set x to any of the following: 

int x = 1;
int x = 21;
int x = 42069;

You can even assign omit the assignment, and just declare the variable: 

int x;

We can also name the variable anything we want, for example: 

int mybeautifulint = 0;
Or even 
int imjustgonnaputrandomstuffhere = 0;

Although Java will accept any name, it makes sense to make the names meaningful and easy to read. For example: 

int length = 3;
int width = 2;
int area = 6;

Another useful convention to follow is to camel-case your variables. In camel-case, the first letter of each word is capitalized. For example, if we had a variable that represents the minimum size of a skyscraper, we can use: 

int minimumSkyscraperSize = 100;
Here are some more examples of camel-case: 
int smallLength = 3;
int smallWidth = 2;
int smallArea = 6;
int largeLength = 20;
int largeWidth = 20;
int largeArea = 400;

The final thing to learn about for variables, are datatypes.

Datatypes

All variables in Java must have a datatype specified. In our previous examples, we always used the int datatype.

However, there are many other datatypes in Java, such as double, char and String. To use these, replace int with the desired datatype. For example: 

double d = 0.1;
char c = 'x';
String s = "hello world!";

char can only store a single ascii character. You have to wrap the value you give it in single quotations ('). For example: 

char c = 'a';
char d = 'z';
char e = '1';
char f = 'A';
char n = '\n';

are all valid, but 

char g = '12';
char h = 'ab';
char i = '.ac.';
char j = "abc";

will not work.

int can store any integer (number without decimals). An int can store any integer, as long as it's greater than or equal to -2147483648 and smaller than or equal to 2147483647

A String can store any sequence of char datatypes. Their values are wrapped in double-quotes ("). For example: 

String s = "Hello World!";
String a = "Ipsum Lorem 123\n";
Are all valid.

Don't worry too much about these constraints, they will become familiar with practice. Later in the course, we will learn much more about datatypes in Java, including all their differences. For now, just remember these 2 datatypes:

int and String.

Basic Arithmetic with Variables

One final note about variables, is that we can do basic arithmetic with them. Consider this code: 

int smallLength = 3;
int smallWidth = 2;
int smallArea = 6;
int largeLength = 20;
int largeWidth = 20;
int largeArea = 400;

Instead of doing the calculations of the area in our head, we can use the * operator to multiply variables: 

int smallLength = 3;
int smallWidth = 2;
int smallArea = smallLength * smallWidth;
int largeLength = 20;
int largeWidth = 20;
int largeArea = largeLength * largeWidth;

We can also multiply values directly, without variables: 

int volume = 2*3*10
And do the same with addition (+), subtraction (-), and division (/): 
int d1 = 5+16;
int d2 = 31;
int totalDistance = d1+d2;
int triangleArea = (5+7)/2;
int age = 2021-2004;

Notice that the order of operations is the same in Java as in normal mathematics. For example, the value of 10+20/2 is 20, while the value of (10+20)/2 is 15

Getting Input

Alright, that was a lot content, let's do something cool with what we learned ... let's interact with the user.

First, let's open up a brand new Java file. To get user input, we will add the following line of code to the top of our file: 

import java.util.Scanner;

This tells Java to bring some external code (in this case, java.util.Scanner) into the program, so that we can use it.

Now, we can use Scanner using the following code: 

Scanner sc = new Scanner(System.in);

Notice that this is similar to declaring variables. In this case, the datatype is Scanner, and we are assigning it the value of new Scanner(System.in). For now, don't worry about this part means. It's enough to know that this will allow us to get input from the console, through the sc variable.

Your code should now look something like this:

import java.util.Scanner;
public class Main{
	public static void main(String[] args){
		//TODO
		Scanner sc = new Scanner(System.in);
	}
}

To use the Scanner, we use the variable sc as follows: 

String s = sc.next();

This stores the user input as a String into String s. We can print the value of s using 

System.out.println(s);

We can also get an int from the console input as follows: 

int in = sc.nextInt();

Using everything we learned and putting it all together, we can create a simple program to get the user's name and age, and output it them: 

import java.util.Scanner;
public class Main{
	public static void main(String[] args){
		//TODO
		Scanner sc = new Scanner(System.in);
		System.out.println("What is your name?");
		String name = sc.next();
		System.out.println("What is your age?");
		int age = sc.nextInt();
		System.out.println("Hello!");
		System.out.println(name);
		System.out.println("You are: ");
		System.out.println(age); //we can also output int datatypes through System.out
		System.out.println("years old!");
	}
}

Notice that besides Strings, we can also output datatypes such as int. In fact, all datatypes are printable through System.out (we will learn more about why later).
